# 紙張輸出估價工具 Paper Pricing Estimator

線上小工具：輸入印製尺寸與數量、勾選要比較的紙張種類，即可用「購物車」的方式加入多個紙張選項，一次比較各項小計與總價。

**線上使用：** https://s46129.github.io/paper-price-tool/

## 功能

- 輸入長、寬、四周留白、數量，即時換算計價才數
- 內建常用尺寸快選（A3 / A4 / A5、4x6 系列相片尺寸 3R–20R），也可以把常用尺寸自行命名儲存起來重複使用
- 紙張清單可直接勾選比較、加到「我的最愛」方便下次快速挑選，名稱與單價皆可直接編輯，也能新增/刪除紙張
- 購物車功能：把設定好的尺寸與勾選的紙張加入購物車，可個別調整數量、移除項目，並自動標示最便宜的選項與總計金額
- 所有清單與購物車內容都存在瀏覽器本機（localStorage），不會上傳到任何伺服器，重新整理頁面也不會消失；但不同瀏覽器/裝置之間資料不會同步

## 計價規則

- 1 才 = 900 平方公分
- 未滿 1 才以 1 才計價，超過 1 才後以每 0.5 才為單位無條件進位
- 單張價格 = 計價才數 × 每才單價；購物車小計 = 單張價格 × 數量

## Data 來源

紙張種類與每才單價的預設清單，整理自 **Avocado Art Lab** 的輸出價格試算表，相關資訊可參考：

- [紙材介紹](https://www.avocado.art/%E7%B4%99%E6%9D%90%E4%BB%8B%E7%B4%B9)
- [輸出方案及須知](https://www.avocado.art/%E8%BC%B8%E5%87%BA%E6%96%B9%E6%A1%88)

頁面上的紙張名稱與價格皆可自行編輯調整，實際價格仍以 Avocado Art Lab 公告或報價為準。

## 本機開發

這是純前端單一 HTML 檔案（`index.html`），沒有任何建置流程或外部相依套件，用瀏覽器直接開啟即可使用；若要部署，把 `index.html` 放到任何靜態網站空間（GitHub Pages、Netlify 等）即可。
